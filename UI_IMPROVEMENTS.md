# UI 개선 아이디어

## 완료

### 1. 메시지 화면 개선
- AI 파싱 결과 구조화 표시 (신뢰도 배지, 거래 유형, 금액, 가맹점, 계정명)
- 상태 탭 한국어화 + duplicate 탭 추가
- 실패 메시지 안내 문구

## 예정

### 2. 분개 배치 승인 ✅
체크박스로 여러 건 선택 후 한 번에 confirm. 전체 선택/해제, 선택 카드 하이라이트.

### 3. 스와이프 제스처 (모바일)
좌/우 스와이프로 reject/confirm. 폰에서 검토하는 앱이므로 유용.

### 4. 대시보드 — 사람별 지출 분리
device_name 기반으로 가족 구성원별 지출 비교 뷰 추가.

### 5. PWA 등록 ✅
manifest + service worker 추가로 모바일 홈 화면에 앱처럼 설치 가능. 정적 자산 캐싱, API는 네트워크 전용.

### 6. 리파스 결과 diff 표시
재파싱 전/후 결과를 나란히 비교. 어떤 필드가 바뀌었는지 시각적으로 확인.
